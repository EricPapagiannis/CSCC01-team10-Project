Valid Commands

help

	no args

	prints information about program usage

update

	no args

	updates system from set up remote databases; generates and
	presents proposed changes to user

showall

	no args

	presents all proposed changes to user

acceptall

	no args

	approves all proposed changes

acceptall2

	no args

	approves all proposed change alternative mode

denyall

	no args

	disregards all proposed changes

status

	no args

	view information about the current system

postponeall

	no args

	sends all proposed changes to blacklist

clearblacklist

	no args

	removes all proposed changes currently in blacklist

stopautoupdate

	no args

	stops the autoupdate daemon

clearrepo

	no args

	clears the configured repo
